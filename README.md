## Commands

### Start the application

```bash
poetry run invoke start
```

### Test


```bash
poetry run invoke test
```

### Test coverage

```bash
poetry run invoke coverage-report
```

### Pylint

```bash
poetry run invoke lint
```

### Refactoring


```bash
poetry run invoke refactor
```
